# Lopa

Lopa is a lightweight, centralized, single-binary network quality measurement and monitoring tool written in Go.

This repository is under active development. The initial goal is to support:

- ICMP ping
- TCP ping (TCPing)
- UDP probe
- TWAMP / TWAMP-light

## Build

```bash
go build ./cmd/lopa
```

## Run

```bash
./lopa --help
```
